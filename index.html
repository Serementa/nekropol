Технология
Как мы читаем стёртые надписи

Многие надписи на каменных крестах XIX века стёрты временем настолько, что не читаются визуально. Но методология 3D-моделирования через Agisoft Metashape позволяет восстановить их через карту высот рельефа.

Методика разработана в рамках проекта «Свод русских надписей» и применяется для памятников даже XVII века. По рекомендации А. А. Зиганшиной, мы адаптируем этот подход для казачьих надгробий Убеженской.

Процесс: 30–50 ортогональных снимков каждого объекта → построение 3D-модели → генерация карты высот → чтение надписей в рельефе поверхности.

Agisoft Metashape

Программа фотограмметрии для построения 3D-моделей по фотографиям. Позволяет создать карту высот, на которой видны рельефные надписи, невидимые невооружённым глазом.

Статья Зиганшиной А. А. и Дынина М. Д.

«Документирование надгробия: современные методы» — обоснование применения 3D-моделирования для чтения надписей на исторических надгробиях.
